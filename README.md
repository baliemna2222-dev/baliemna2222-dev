## About

I'm a 2nd-year software engineering student who got tired of code that stays on screens.  
So I started building things that move, blink, and sometimes catch fire (affectionately).

Right now I'm somewhere between full-stack dev and robotics tinkerer — writing Angular frontends in the morning and soldering servo wires at midnight.

```
curiosity → prototype → break it → fix it → learn → repeat
```

## 🔧 Currently Building

**Quadruped Robot** — An 8-servo walking robot driven by an ESP32, controlled over Bluetooth.  
Wrangling gait sequences, power rail issues, and 3D-printed parts in PETG.  
Status: *it walks. mostly.*
